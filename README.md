# ncnn_faster-mobile-retinaface

移动端快速人脸检测模型是基于RetinaFace的优化去掉stride8以及stride32和stride16的landmark
在CPU位Intel(R) Pentium(R) CPU G2020 @ 2.90GHz(2900 MHz)的设备中人脸检测可达到40ms/帧

# 开发环境

win7
vs2015
